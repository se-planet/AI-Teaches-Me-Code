## ✅ 一、数组（Array）

> 📅 学习时间：2026-07-01 ~ 2026-07-02
> 📂 代码：[01_数组/lesson_array.py](01_数组/lesson_array.py)

### 1.1 核心概念

| 特性 | 说明 |
|------|------|
| **存储方式** | 一块连续的内存空间，像一排紧挨的停车位 |
| **索引访问 O(1)** | 知道首地址 + 偏移量，直接算出任意元素位置 |
| **插入/删除 O(n)** | 中间插入要挪动后面所有元素 |
| **查找 O(n)** | 最坏情况要遍历整个数组 |

### 1.2 时间复杂度速查

| 操作 | 复杂度 | 原因 |
|------|:--:|------|
| 按索引读 | O(1) | 连续内存，直接计算 |
| 按索引写 | O(1) | 同上 |
| 尾部插入 `append` | O(1) | 末尾有空位 |
| 中间插入 `insert` | O(n) | 后面元素要后移 |
| 按值查找 | O(n) | 一个个遍历 |
| 删除 | O(n) | 后面元素要前移 |

### 1.3 核心技巧：双指针

两个指针从不同位置出发，向中间（或同向）移动，避免额外空间。

**常见变体：**
- **左右指针**：反转数组、两数之和
- **快慢指针**：去重、移动零

**模板：**
```python
left, right = 0, len(arr) - 1
while left < right:
    arr[left], arr[right] = arr[right], arr[left]  # 交换
    left += 1
    right -= 1
```

### 1.4 两数之和（Two Sum）详解

> LeetCode 第 1 题，新手必刷

**题目：** 找数组中两数之和等于 target 的索引。

**暴力法 O(n²)：** 每两个数都加一遍试试。

```python
for i in range(len(nums)):
    for j in range(i + 1, len(nums)):   # 从 i+1 开始，避免重复用同一元素
        if nums[i] + nums[j] == target:
            return [i, j]
```

**哈希法 O(n)：** 把加法变成减法，算搭档，查字典。

```python
seen = {}  # {数值: 索引}
for i, num in enumerate(nums):
    complement = target - num      # 算搭档
    if complement in seen:         # 搭档在吗？
        return [seen[complement], i]
    seen[num] = i                  # 不在，把自己存下来
```

**关键：先查后存。** 如果先 `seen[num] = i` 再查，`[3, 3]` target=6 会返回 `[0, 0]`（找到了自己）。

**两种解法对比：**

| | 暴力 | 哈希 |
|------|------|------|
| 思路 | 两两相加试 | 算搭档，翻笔记本 |
| 时间 | O(n²) | O(n) |
| 空间 | O(1) | O(n) 多一个字典 |
| 比喻 | 全班两两握手 | 前台查登记表 |

### 1.6 Python 注意点

- `list.sort()` 返回 `None`（原地排序），`sorted(list)` 返回新列表
- 切片是左闭右开：`arr[a:b]` 含 a 不含 b
- `arr2 = arr1` 是浅拷贝，改一个另一个也变 → 用 `arr1[:]` 或 `arr1.copy()`
- 遍历时不要修改数组本身
