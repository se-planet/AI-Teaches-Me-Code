## 🔥 二、链表（Linked List）

> 📅 学习时间：2026-07-03 ~ 进行中
> 📂 代码：[02_链表/lesson_linked_list.py](02_链表/lesson_linked_list.py)

### 2.1 链表 vs 数组

| | 数组 | 链表 |
|------|------|------|
| 内存 | 连续一块 | 散落各处，指针相连 |
| 索引访问 | O(1) 瞬间 | O(n) 必须从头走 |
| 插入/删除 | O(n) 挪位置 | O(1) 改指针就行（头部） |
| 查找 | O(n) | O(n) |
| 比喻 | 停车场一排紧挨车位 | 火车，车厢用钩子相连 |

### 2.2 节点结构

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val      # 数据
        self.next = next    # 指向下一个节点的指针
```

### 2.3 基本操作

| 操作 | 复杂度 | 关键点 |
|------|:--:|------|
| 头部插入 | O(1) | 新节点 → 旧头 |
| 尾部插入 | O(n) | 走到末尾再挂 |
| 删除 | O(n) | 先找到，跳过它 |
| 遍历 | O(n) | `while cur:` |

### 2.4 核心技巧

**反转链表（四步公式）：**

```python
prev = None; cur = head
while cur:
    nxt = cur.next      # ① 记后路
    cur.next = prev     # ② 箭头调头（核心）
    prev = cur          # ③ prev 前进
    cur = nxt           # ④ cur 前进
return prev
```

**快慢指针：**

| 应用 | 技巧 |
|------|------|
| 找中点 | 快走两步、慢走一步 |
| 判环 | 快慢指针相遇 = 有环 |

### 2.5 关键原则

- **head 不动，cur 走路** — head 是门牌号，丢了就找不到链表了
- **改指针前先记后路** — `nxt = cur.next` 后再改 `cur.next`
- **空链表先判断** — `if not head: return`

### 2.6 Python 注意点

- 链表节点是对象，`cur.next` 读是取值，`cur.next = xxx` 是改箭头
- `while cur` 遍历每个节点，`while cur.next` 停在最后一个节点
- LeetCode 上 ListNode 已定义好，本地需自己定义

## ✅ 一、数组（Array）

> 📅 学习时间：2026-07-01 ~ 2026-07-02
> 📂 代码：[01_数组/lesson_array.py](01_数组/lesson_array.py)

### 1.1 核心概念

| 特性 | 说明 |
|------|------|
| **存储方式** | 一块连续的内存空间，像一排紧挨的停车位 |
| **索引访问 O(1)** | 知道首地址 + 偏移量，直接算出任意元素位置 |
| **插入/删除 O(n)** | 中间插入要挪动后面所有元素 |
| **查找 O(n)** | 最坏情况要遍历整个数组 |

### 1.2 时间复杂度速查

| 操作 | 复杂度 | 原因 |
|------|:--:|------|
| 按索引读 | O(1) | 连续内存，直接计算 |
| 按索引写 | O(1) | 同上 |
| 尾部插入 `append` | O(1) | 末尾有空位 |
| 中间插入 `insert` | O(n) | 后面元素要后移 |
| 按值查找 | O(n) | 一个个遍历 |
| 删除 | O(n) | 后面元素要前移 |

### 1.3 核心技巧：双指针

两个指针从不同位置出发，向中间（或同向）移动，避免额外空间。

**常见变体：**
- **左右指针**：反转数组、两数之和
- **快慢指针**：去重、移动零

**模板：**
```python
left, right = 0, len(arr) - 1
while left < right:
    arr[left], arr[right] = arr[right], arr[left]  # 交换
    left += 1
    right -= 1
```

### 1.4 两数之和（Two Sum）详解

> LeetCode 第 1 题，新手必刷

**题目：** 找数组中两数之和等于 target 的索引。

**暴力法 O(n²)：** 每两个数都加一遍试试。

```python
for i in range(len(nums)):
    for j in range(i + 1, len(nums)):   # 从 i+1 开始，避免重复用同一元素
        if nums[i] + nums[j] == target:
            return [i, j]
```

**哈希法 O(n)：** 把加法变成减法，算搭档，查字典。

```python
seen = {}  # {数值: 索引}
for i, num in enumerate(nums):
    complement = target - num      # 算搭档
    if complement in seen:         # 搭档在吗？
        return [seen[complement], i]
    seen[num] = i                  # 不在，把自己存下来
```

**关键：先查后存。** 如果先 `seen[num] = i` 再查，`[3, 3]` target=6 会返回 `[0, 0]`（找到了自己）。

**两种解法对比：**

| | 暴力 | 哈希 |
|------|------|------|
| 思路 | 两两相加试 | 算搭档，翻笔记本 |
| 时间 | O(n²) | O(n) |
| 空间 | O(1) | O(n) 多一个字典 |
| 比喻 | 全班两两握手 | 前台查登记表 |

### 1.6 Python 注意点

- `list.sort()` 返回 `None`（原地排序），`sorted(list)` 返回新列表
- 切片是左闭右开：`arr[a:b]` 含 a 不含 b
- `arr2 = arr1` 是浅拷贝，改一个另一个也变 → 用 `arr1[:]` 或 `arr1.copy()`
- 遍历时不要修改数组本身
